# VuePress 模板

这是一个使用 VuePress 2.0 构建的文档站点模板。VuePress 是一个以 Markdown 为中心的静态网站生成器，使用 Vue 构建，非常适合用于创建文档站点。

## 特性

- **简洁至上**：专注于内容创作，使用 Markdown 轻松编写文档
- **Vue 驱动**：在 Markdown 中使用 Vue 组件
- **高性能**：静态 HTML 生成 + SPA 客户端体验
- **可定制**：支持自定义主题和插件
- **响应式设计**：适应各种屏幕尺寸
